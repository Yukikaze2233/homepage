## 本地预览经验
- `file://` 直接双击打开 `index.html` 不可靠
- 浏览器可能分别复用 HTML 和图片缓存，导致出现：
  - 旧 HTML + 新图片
  - 新 HTML + 旧图片
- 后续统一使用本地 HTTP 服务预览：

```bash
cd "/home/yukikaze/Documents/workspace/homepage"
python3 -m http.server 8000
```

然后打开：

```text
http://127.0.0.1:8000/
```
